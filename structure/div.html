DIV Tags

DIV tags can be used when you need to attach an identifying attribute to a tag.

DIV and Span tags are useless without an attached class attribute. A class can have any name and you can have as many as you want.

To target a div class in the CSS, you always want to begin the selector with a period. This tells the CSS that your selector is a class and not a standard tag, which is an important distinction.
