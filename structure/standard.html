Standard Tags

Standard tags are additional tags that help define content on a website. There is a standard tag for headers, nav, section, footer, and more.

It's best to use standard tags to establish structure because browsers recognize these tags. When the browser knows where and what your navigation is, it can communicate that information to search engines, screen readers, and other web tools.
